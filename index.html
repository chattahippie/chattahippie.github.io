Leviathan

Leviathan is a object-oriented programming language designed to prototype and draft ideas.

Its highly versatile grammar allows the user to quickly jot down ideas in an almost pseudo-code syntax, then see their ideas execute immediately. Ideas can be expressed as verbosely as needed, or using the shortcut syntax built into Leviathan to reduce the time needed to move a program from your thoughts into the computer.

Leviathan supports features such as lambda functions, anonymous objects, python-like for loop iteration, and include statements to make it a powerful language. Check out its repository here to see the source and get started!

iMessage Commands

iMessage Commands is a proof of concept project to show how a user can use Apple's iMessage to interface with command line programs. To show how this works, I have created an AppleScript event handler to allow users away from a terminal to access the popular programs fortune and ansiweather.

Simply messaging !fortune or !weather <city> to an iMessage account with the handler set to this script will elicit an immediate response to the user with the response of the program.

This powerful concept can be applied to just about any command line program, which can vastly increase a user's productivity from a phone. Using this proof of concept, it is easy to configure new commands while keeping the data inputted from the user sanitized. Everything needed to get started can be found here.
